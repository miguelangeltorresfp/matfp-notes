# TEMPLATE LITERALS

* by Dylan Israel on Scrimba - [LINK](https://scrimba.com/p/p4Mrt9/c4vJdha)

---------------------------------

* OLD WAY ->

```bash
let word1 = 'Dylan';
let word2 = 'Israel';

const fullName = word1 + ' ' + word2;
```

* NEW WAY ->

```bash
let word1 = 'Dylan';
let word2 = 'Israel';

const fullName = `${word1} ${word2}`;
```

---------------------------------

* SUMM VARIABLES

```bash
let num1 = 2;
let num2 = 3;

const summ = `${num1 + num2}`;
```

---------------------------------

* MULTILINE STRINGS

* OLD WAY ->

```bash
const fullName = 'Hello \n' + 'world';
```

* NEW WAY ->

```bash
const newWay = `${word1}
${word2}`;
```
