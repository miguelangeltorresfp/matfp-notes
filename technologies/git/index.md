# BASICS
* git init
* git remote -v  
  Lists a list of remotes for a git repository.
* git add <file\>   
  Add files to the index
* git reset <file\>  
  Remove files from the index
* git rm --cached <file\>
  To unstage files from the index    
* git checkout -- <file\>
  To discard changes in working directory
